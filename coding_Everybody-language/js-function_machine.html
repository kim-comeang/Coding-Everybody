함수의 입력, 출력

함수의 입력

function - 기능, 작용 (함수)
함수 = 상자

return이 나오면 함수를 종료
return 'hello';
세미콜론 전까지 있는 값을 함수의 출력값으로 반환한다.

함수의 출력

              함수    번수
function get_argument(arg) {
    return arg;
}
// arg 변수 함수 괄호안에 1,2 를 넣어 변수에 담아 출력. 
get_argument(1);    // 함수 실해시 1;
get_argument(2);    // 함수 실해시 2;

위 변수를 매개변수,파라미터(parameter) 라고 부른다.
함수 괄호안에 값을 인자(argument) 라고 부른다.

엄격하게 나누면 값 자체는 인자, 그 값을 담는 변수는 매개변수, 파라미터라고 부른다.
