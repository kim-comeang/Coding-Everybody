객체와 반복문

배열은 저장된 데이터들이 순서가 있다.
객체에 저장된 값은 순서가 없다. key와 value가 있다.

배열에 있는 값을 가져올때는 for/in 문이라는것을 이용한다.

var grades = {'a': 10, 'b': 6, 'c': 80};
grades 라는 각각 숫자(점수)를 가지고 있는 객체를 만들었다.

for(key in grades) {
1. key값이 들어가는데 이것은 약속이다.
2. in을 중심으로 앞에 key를, 뒤에 객체를 넣는다.
3. grades라는 변수가 담고있는 값들을 반복문이 실행될때마다 하나씩 가져와서 key에 담는다. (key값을)

    document.write("key: " + key + " value: " + grades[key] + "
");
};
