비교

a = 1 // a라는 변수(바뀌는 값)에 1이라는 값(상수: 바뀌지 않는 값)을 대입 한다.

= 대입 연산자: 좌항에 있는 값을 우항에 대입한다.

== 비교연산자 (<, >, >=, <=): boolean(true, false)값 반환

동등 연산자, 일치 연산자

== : 동등 연산자 (equal(동일한) operator)

=== : 일치 연산자 (strict(엄격한) operator) 동등 연산자보다 일치연산자를 사용할것을 권장 한다.

null(값이 없다), undefined(값이 정의되지 않았다) 특수한 형식의 데이터

boolean: true, false - 자바스크립트 데이터 타입중 하나

null, undefined 각각 데이터 타입중 하나

자바스크립트는 true를 숫자 1로 간주함, 1을 제외한 숫자를 false로 간주함

일치 연산자로 비교하면 false

https://dorey.github.io/JavaScript-Equality-Table/

!= 느낌표는 부정을 의미

!== 정확하게 같지 않다는 의미

>= 좌항이 우항보다 크거나 같다. (<=는 반대 의미)
