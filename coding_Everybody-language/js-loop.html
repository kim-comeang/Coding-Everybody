반복문 - loop (iterate)

while과 for가 있다.

while 문

while (조건) { true에서 false가 될때까지
    반복해서 실행할 코드
};

- 무한루프
while (true) {
    document.write("Hello World. 
");
}

for 문

초기화, 반복 조건(불린 값), i++(+1)
i의 값이 10보다 작을때까지 실행

i는 iterate의 앞자를 따서 관용적으로 사용한다.
i = i + 1;
i++;    // 기존의 값부터 사용
++i;    // 기존의 값을 증가시키면서 사용

반복문의 제어

break - 반복문 종료

continue - 그 순간 종료

    반복문 안에 조건문이, 조건문 안에 반복문이 들어갈수 있다.

반복문의 중첩

1 반복문 1회당 j 반복문 10회 실행

j 실행 = 0 ~ 9 > 총 10회
